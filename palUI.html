Metapalindrome Finder

Find the length of the longest contiguous substring that appears both forward and backward in an input string. The forward and backward substrings must not be adjacent or overlap. For example:

• Given the input string RECURSION, the solution is 1

• Given the input string REDIVIDE, the solution is 3

• Given the input string DYNAMICPROGRAMSMANYTIMES, the solution is 4
